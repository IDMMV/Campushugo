# CampusHugo Pro - Fase 1 Completa

Esta versión corrige el problema de archivos falsos `.txt`: los botones de plantillas ahora no descargan placeholders, sino que envían solicitud por WhatsApp hasta que se creen recursos reales.

## Incluye
- Página principal moderna.
- Apariencia mejorada en celular.
- Menú responsive.
- Catálogo de cursos.
- Plantillas en modo solicitud.
- Herramientas básicas funcionales.
- Login demo.
- Panel admin demo.
- PWA básica.
- SQL inicial para Supabase.
