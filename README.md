# CampusHugo Pro - Motor Interactivo v4

Versión oficial con ruta Excel ampliada a 300 lecciones y motor interactivo inicial.

## Incluye
- 300 lecciones organizadas en 5 niveles.
- Tutor con voz usando SpeechSynthesis del navegador.
- Simulador de Excel básico.
- Práctica guiada por celda y fórmula.
- Quiz, XP y progreso local.
- Constructor visual inicial de lecciones.
- SQL base para Supabase.

## Cómo probar
1. Sube todo el contenido a GitHub Pages o Vercel.
2. Abre la web.
3. Entra a Academia Excel.
4. Toca Entrar en cualquier lección.
5. Usa Escuchar guía, practica B5, escribe la fórmula y completa el quiz.
