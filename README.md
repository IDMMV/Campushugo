# CampusHugo - Motor Interactivo Definitivo v1

Esta entrega es la primera versión oficial del motor interactivo.

## Incluye
- Reproductor de lecciones por bloques.
- Tutor IA simulado con voz del navegador.
- Ejercicios tipo Excel.
- Validación de fórmula.
- Mini quiz.
- XP y progreso local.
- Constructor visual inicial.
- SQL base para Supabase.

## Cómo probar
1. Sube todo el contenido a GitHub Pages o abre `index.html`.
2. Entra a Aprender.
3. Abre una lección.
4. Usa `Escuchar guía`.
5. Completa la práctica y guarda progreso.
